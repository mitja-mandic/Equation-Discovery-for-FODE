# About SLAIF short trainings

Alongside the longer workshop books published in this collection, SLAIF runs a
series of **short trainings**: compact, focused sessions that each target a
single practical skill rather than a full multi-chapter course. They are meant
to get researchers, students, and engineers productive quickly on one specific
topic, without committing to a long course.

The trainings cluster around the themes that recur across SLAIF's educational
work package, such as the following:

- **High Performance Computing (HPC) and supercomputing access** — getting onto
  shared clusters, submitting jobs, and using the SLING infrastructure
  effectively.
- **Containers and reproducibility** — packaging research software with tools
  such as Apptainer so that it runs the same way everywhere.
- **AI and agentic programming** — using modern AI assistants and orchestrated,
  agentic coding workflows in day-to-day research and development.
- **Practical research-computing skills** — the smaller command-line, tooling,
  and environment topics that support the above.

Each short training is self-contained. You can follow a single session on its
own, and the supporting materials — slides, handouts, example files, and any
recordings — are gathered together in one shared location.

# All resources

The complete set of materials for the short trainings is published in a single
shared folder:

- [Short trainings — all resources](https://mapa.sling.si/s/wAQQrqjsaYcswMm)

Open the link to browse the materials for every session. The folder is the
authoritative source for the short trainings - i.e., lectures, webinars or short workshops.

<Sidenote>
**Looking for the longer courses?** The multi-chapter workshop books, such as the
Apptainer workshop and the Orchestrated Agentic Programming materials, live
elsewhere in the SLAIF trainings collection.
</Sidenote>
