Uploaded document: "MacroPCA: An All-in-One PCA Method Allowing for Missing Values as Well as Cellwise and Rowwise Outliers" (Hubert, Rousseeuw, and Van den Bossche, 2019).

Document summary:
MacroPCA is a robust principal component analysis method for data containing missing values, cellwise outliers, and rowwise outliers simultaneously. It first uses DetectDeviatingCells to identify suspicious cells and form initial imputations. It then combines iterative PCA-style imputation with ROBPCA-based robust subspace estimation, selects an appropriate number of components, reweights non-outlying rows, and uses robust estimation within the selected subspace. The method produces diagnostics, including outlier and residual maps, to identify problematic rows, cells, and variables. The paper demonstrates that it remains useful when classical PCA and methods designed for only one type of contamination fail.
